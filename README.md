# Gemini-for-Discord

Асинхронный Discord-бот, интегрированный с нейросетью Google Gemini (`gemini-3.1-flash-lite`), предназначенный для генерации текста, анализа истории чата, суммаризации обсуждений и код-ревью.

---

## Что это такое?
Это Discord-бот, написанный на Python, который предоставляет пользователям доступ к возможностям языковых моделей Google Gemini непосредственно в текстовых каналах сервера. Бот оптимизирован для работы в условиях региональных ограничений за счет встроенной поддержки прокси-серверов и содержит отказоустойчивую архитектуру обработки ошибок API.

---

## Технические особенности:
* **Поддержка прокси:** Весь трафик к серверам Gemini и Discord API маршрутизируется через прокси для обхода блокировок.
* **Потокобезопасность:** Блокирующие вызовы SDK Gemini вынесены в отдельные потоки (`asyncio.to_thread`), что предотвращает зависание основного event loop бота.
* **Валидация вывода:** Системный промпт принудительно ограничивает размер ответа до 1950 символов, исключая ошибки падения Discord из-за превышения лимитов длины сообщения.

## Discord-бот

### Доступные команды бота:
* **`/help`** — Вывод интерактивной справки по командам проекта.
* **`/ai <prompt>`** — Простое обращение к искусственному интеллекту. Задайте свой вопрос!
  * *Пример использования:* `/ai Сколько весит слон?`
* **`/context_ai <prompt>`** — Анализ контекста. Бот поможет проанализировать конец вашего диалога (последние 100 сообщений) и ответить на ваш вопрос.
  * *Пример использования:* `/context_ai Найди в диалоге ссылку на домашнее задание №3`
* **`/retell <N>`** — Саммаризация последних сообщений. Бот пересказывает, о чём шёл диалог в последние `N` сообщений (по умолчанию 50).
  * *Пример использования:* `/retell 50` (или без параметров: `/retell`)
* **Контекстная команда «Объяснить простым языком»** — Даёт пояснение к сложной информации, будь то код, формулы или задачи. Чтобы использовать, нажмите правой кнопкой мыши на нужное сообщение, затем выберите **Приложения > GeminiAIHelperBot > Объяснить простым языком**.

## ВК-бот

### Доступные команды бота:
* **`/help`** — Вывод интерактивной справки по командам проекта.
* **`/ai <prompt>`** — Простое обращение к искусственному интеллекту. Задайте свой вопрос! Поддерживает контекст обсуждения, если отправлена в качестве ответа (reply) на другое сообщение.
  * *Пример использования:* `/ai Сколько весит слон?`
* **`/context_ai <prompt>`** — Анализ контекста. Бот поможет проанализировать конец вашего диалога (последние 100 сообщений) и ответить на ваш вопрос.
  * *Пример использования:* `/context_ai Найди в диалоге ссылку на домашнее задание №3`
* **`/retell <N>`** — Саммаризация последних сообщений. Бот пересказывает, о чём шёл диалог в последние `N` сообщений.
  * *Пример использования:* `/retell 50`
* **`/explain`** — Даёт пояснение к сложной информации, будь то код, формулы или задачи. Работает, только если ответить на какое-то сообщение этой командой.

## Что нужно для работы?

Для работы проекта необходимо настроить конфигурацию среды окружения. В корневой директории проекта должен быть создан файл `.env` (шаблон находится в `.env.example`) со следующими переменными:

```env
#ДЛЯ РАБОТЫ DISCORD-БОТА:
DISCORD_TOKEN=YOUR_DISCORD_TOKEN     # Токен вашего бота из Discord Developer Portal

#ДЛЯ РАБОТЫ ВК-БОТА:
VK_GROUP_ID=YOUR_VK_GROUP_ID     # Числовой ID группы вашего бота в ВК.
VK_TOKEN=YOUR_VK_TOKEN     # API-ключ вашего бота в ВК

#ДЛЯ ОБРАЩЕНИЯ К GEMINI:
GEMINI_TOKEN=YOUR_GEMINI_API_KEY     # Ключ API для доступа к Google Gemini

#ДЛЯ ПОЛКЛЮЧЕНИЯ ЧЕРЕЗ ПРОКСИ-СЕРВЕР. УДАЛИТЕ ЭТУ СТРОКУ, ЕСЛИ УВЕРЕНЫ, ЧТО ВАШ СЕРВЕР НЕ ЗАТРОНУТ РЕГИОНАЛЬНЫМИ ОГРАНИЧЕНИЯМИ GOOGLE И DISCORD.
PROXY=YOUR_PROXY                     # URL-адрес прокси-сервера (например, http://ip:port)
```

## Как это установить и запустить?

Инструкция для развертывания проекта с нуля:

### Шаг 1: Клонирование и подготовка директории
Убедитесь, что на сервере/компьютере установлен Python 3.10 или выше. Перейдите в корневую папку проекта.

### Шаг 2: Настройка виртуального окружения
Инициализируйте и активируйте изолированное виртуальное окружение:

```bash
# Создание окружения (папка .venv добавлена в .gitignore)
python -m venv .venv

# Активация для Linux / macOS:
source .venv/bin/activate

# Активация для Windows (Command Prompt):
.venv\Scripts\activate
```

### Шаг 3: Установка зависимостей
Установите внешние библиотеки, необходимые для работы клиентов Discord и Gemini:
```bash
pip install discord.py google-genai python-dotenv vkbottle
```

### Шаг 4: Конфигурация окружения
Создайте файл .env на основе примера:

```bash
cp .env.example .env
```

Заполните созданный файл .env актуальными токенами и адресом прокси.

### Шаг 5: Запуск бота
Убедитесь, что вы находитесь в папке с проектом, и запустите бота.

Запуск Дискорд-бота:
```bash
python main.py
```
Запуск ВК-бота:
```bash
python VK_Bot.py
```

## Сокращенная инструкция для быстрого запуска напрямую в систему (без виртуального окружения):

### Шаг 1: Установка необходимых библиотек
Откройте терминал или командную строку и установите зависимости одной командой:

```bash
pip install discord.py google-genai python-dotenv vkbottle
```

### Шаг 2: Настройка конфигурации
Переименуйте файл .env.example в .env и откройте его в текстовом редакторе. Заполните файл .env актуальными токенами и адресом прокси.

### Шаг 3: Запуск проекта
Убедитесь, что вы находитесь в папке с проектом, и запустите бота.

Запуск Дискорд-бота:

```bash
python main.py
```
Запуск ВК-бота:

```bash
python VK_Bot.py
```
